WifiPassUpdater
===============

## Setup

```
npm install
```


## Configure

```
touch data/config.json
```

```
{
    "url": "your_wifi_router_address",
    "user": "basic_authentication_username",
    "pass": "basic_authentication_password",
    "wifi_pass": "new_wifi_pass"
}
```


## Usage

```
$ npm run update-wifi-pass
```
